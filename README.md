
## Installation

To install dependencies for this project run

```bash
  npm install
```

## Start

To start this project on localhost run

```bash
  node index.js
```

## Main .JS Files

The main .js files

### Entry Point
```bash
  cd ./index.js
```
### Controllers/Express Route Files
```bash
  cd ./Controller/
```
### Route function handler files
```bash
  cd ./Services
```
